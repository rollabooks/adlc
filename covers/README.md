# Cover Amazon KDP

Questa cartella contiene le cover per la pubblicazione su Amazon KDP.

Vedi [specs.md](specs.md) per le specifiche tecniche complete.

## Struttura

- `assets/` — Elementi grafici condivisi (logo, sfondi, pattern)
- `templates/` — Template sorgente editabili (PSD, AI, SVG, GIMP)
- `<lang>/` — Cover finali per ogni lingua
  - `ebook-cover.png` — Cover eBook Kindle (1600×2560 px)
  - `print-cover.pdf` — Cover completa cartaceo (fronte + dorso + retro)
