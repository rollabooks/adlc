# Idee e Brainstorming

Questa cartella contiene idee, appunti e brainstorming per il libro.

## Come usare

Crea un file Markdown per ogni idea o argomento:

```
docs/ideas/
  api-design-patterns.md
  error-handling-strategies.md
  performance-optimization.md
```

Lo skill **writer** (`/writer`) leggerà questi file e li elaborerà in capitoli strutturati dentro `md/<lang>/`.
